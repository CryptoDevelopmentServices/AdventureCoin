AdventureCoin
=====================================

AdventureCoin aims to encourage micro-payments
for Bitcoin holders by providing a secure and
sustainable open source community and
ecosystem of users, developers and
miners by means of a hardfork
of the Bitcoin blockchain.

https://adventurecoin.org

What is AdventureCoin?
----------------

AdventureCoin is a digital currency forked from [Bitcoin](https://github.com/bitcoin/bitcoin) that enables micro-payments to anyone, anywhere in the world. AdventureCoin uses peer-to-peer technology to operate
with no central authority: managing transactions and issuing money are carried
out collectively by the network. AdventureCoin is the name of open source
software which enables the use of this currency.

For more information, as well as an immediately useable, binary version of
the AdventureCoin software, see https://adventurecoin.org, or read the
[whitepaper](https://adventurecoin.org/assets/ADVC_Whitepaper_v1.pdf).
